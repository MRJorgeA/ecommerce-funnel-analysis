# Ecommerce Funnel Analysis

## Overview
Proyecto de análisis con SQL enfocado en el embudo de conversión y la retención en e-commerce. Identifiqué caídas críticas en el funnel, diferencias por país y patrones de retención por cohortes para generar insights accionables que ayuden a mejorar conversión y permanencia de usuarios.

## Objective
Analizar el recorrido del usuario dentro del funnel de conversión para detectar en qué etapas se pierden más usuarios y evaluar su retención en el tiempo.

## Tools
- SQL
- Funnel Analysis
- Cohort Analysis
- KPI Analysis
- Excel

## Process
- Extracción y análisis de eventos de usuario
- Medición de conversión por etapa
- Segmentación de resultados por país
- Evaluación de retención por cohortes
- Resumen ejecutivo de hallazgos clave

## Key Findings
- Se detectó una caída importante entre la selección de producto y el agregado al carrito
- Se observaron diferencias relevantes en conversión entre países
- La retención temprana mostró puntos críticos de abandono

## Business Value
Este análisis permite identificar oportunidades para optimizar la experiencia del usuario, mejorar la conversión y fortalecer la retención.
